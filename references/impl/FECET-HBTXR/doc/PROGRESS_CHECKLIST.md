# Progress Checklist

## Scope

This checklist compares the current `FECET-HBTXR` repository against the agreed integration plan:

- FACET-style event representation and ellipse supervision
- `HBTXR_v3_0` hybrid tracker, 2-stage training, and host-side Search/Track FSM
- Grounded-SAM based FACET-style dataset preparation from raw EV-Eye data
- Linux `sh` execution path
- Annotation overlay preview and project documentation

## Status Legend

- `[x]`: implemented and verified in repository-level tests
- `[-]`: implemented in code, but not yet validated on the full real dataset / full training run
- `[ ]`: not done yet
